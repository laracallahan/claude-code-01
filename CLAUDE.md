## Commands

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Run tests
npm test

# Run a single test file
npm test -- path/to/file.test.ts

# Lint
npm run lint

# Type check
npm run typecheck
```

> Update these commands once the project is initialized (e.g., `npm create vite@latest` or `npx create-react-app`).
